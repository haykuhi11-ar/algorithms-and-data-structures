🟢 AVL Tree Project

📌 Description

This project implements an AVL Tree — a self-balancing binary search tree.

An AVL tree ensures that the height difference between the left and right subtrees of any node does not exceed 1, providing fast and efficient search, insert, and delete operations.

⚡ Features

✅ Insert nodes

✅ Delete nodes

✅ Search values

✅ Automatic tree balancing

✅ Traversals:

Inorder (Left → Root → Right)

Preorder (Root → Left → Right)

Postorder (Left → Right → Root)


-----------

| Operation | Time Complexity |
| --------- | --------------- |
| Search    | O(log n)        |
| Insert    | O(log n)        |
| Delete    | O(log n)        |

-----------
